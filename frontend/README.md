# ProteinIK frontend

React + Three.js (via @react-three/fiber) dashboard for the ProteinIK
backend. See the top-level [`../README.md`](../README.md) for full setup
instructions covering both the backend and this frontend.

Quick start:

```bash
npm install
npm run dev
```

Expects the backend running at `http://localhost:8000` by default
(override with `VITE_API_BASE`, see top-level README).
